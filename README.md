# vnl-2024-clustering-analysis-mvp

Este projeto é um MVP de clusterização não supervisionada aplicada ao dataset da VNL Masculino de 2024. O objetivo deste trabalho é investigar até que ponto a estrutura geométrica intrínseca das porcentagens de volume de jogo dos atletas é capaz de reconstruir as cinco posições táticas do voleibol. Para isso, comparam-se três algoritmos de clusterização não supervisionados fundamentados em princípios distintos (K-Means, DBSCAN e clusterização hierárquica), avaliando a qualidade dos agrupamentos pelo Adjusted Rand Index (ARI) e pelo Silhouette Score, a fim de medir quão bem cada um recupera os grupos. Este MVP é uma evolução direta do trabalho anterior de análise exploratória sobre o mesmo dataset.

O dataset está disponível publicamente no Kaggle: [VNL 2024 Men's Stats](https://www.kaggle.com/datasets/jonathanpmoyer/vnl-2024-mens-stats)
